# My New Project

This repo contains my ML experiments developed in Google Colab with Drive persistence and GitHub versioning.

## Structure
- `notebooks/` — interactive exploration and experiments
- `src/` — training/inference code
- `models/` — saved models (small) or tracked with Git LFS
- `data/` — local/Drive data (not committed)
- `scripts/` — helper scripts (e.g., download model)
- `requirements.txt` — pinned dependencies for reproducibility
- `model_card.md` — short documentation about current model
